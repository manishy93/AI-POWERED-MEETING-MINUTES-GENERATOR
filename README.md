# 🎯 AI-Powered Meeting Minutes Generator (Audio to Summary)

## 📌 Overview
This project converts meeting audio into structured meeting minutes including:
- Summary
- Key Points
- Action Items

It uses speech recognition to transcribe audio and basic NLP logic to generate insights.

---

## 🚀 Features
- 🎧 Convert audio (.mp3) to text
- 📝 Generate meeting summary
- 📌 Extract key discussion points
- ✅ Identify action items automatically
- ⚡ Works offline (no API required)

---

## 🛠️ Tech Stack
- Python
- Whisper (Speech-to-Text)
- FFmpeg
